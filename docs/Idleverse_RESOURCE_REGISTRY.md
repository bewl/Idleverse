
# Idleverse – Resource Registry

## Purpose

This document defines the **core resource catalog** for Idleverse.

It establishes the **primary economic backbone of the game** so that:

- systems use consistent resources
- AI-generated features integrate properly
- progression tiers remain balanced
- the economy scales predictably

Each resource includes:

- tier
- category
- typical source
- typical usage

These are **design references**, not final balancing numbers.

---

# Tier Model

Idleverse resources follow a **five tier progression structure**.

| Tier | Category | Role |
|----|----|----|
| Tier 1 | Raw Materials | Early game extraction |
| Tier 2 | Processed Materials | Basic manufacturing |
| Tier 3 | Advanced Components | Midgame infrastructure |
| Tier 4 | Exotic Materials | Late game technologies |
| Tier 5 | Cosmic Materials | Endgame progression |

---

# Design Guidelines

When introducing new resources:

### Prefer

- technological names
- clear production chains
- multiple uses

### Avoid

- single-use resources
- fantasy terminology
- overly complex chains

---

# Final Principle

Resources should form **interlocking production webs** that encourage:

- strategic planning
- specialization
- long-term progression

Idleverse’s economy should feel like a **massive automated industrial ecosystem spanning an entire galaxy**.
